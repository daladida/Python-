## 编译与解释
编译：将高级语言源代码转换成目标代码（机器语言），程序便可执行  
解释：将源代码逐条转化成目标代码同时逐条运行，每次运行程序都需要源代码和解释器  
静态语言：通过编译执行的编程语言，如C，Java等  
脚本语言：通过解释执行的编程语言，如Python，JavaScript等  
## Python语法元素
注释 输入 缩进 输出 变量 分支 常量 循环 表达式  
缩进是Python语言中表明程序框架的唯一手段（1个缩进 = 4个空格）
